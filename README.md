# KK1 – Steam Games Dataanalys

En dataanalys av över 120 000 spel publicerade på Steam.
Gjord som del av kursen Artificiell Intelligens – programmering Python (MAI25MA).

## Dataset

Datasetet kommer från Kaggle:
https://www.kaggle.com/datasets/fronkongames/steam-games-dataset

För att köra notebooken:
1. Ladda ner `games.csv` från länken ovan
2. Skapa en mapp som heter `data/` i projektmappen
3. Lägg `games.csv` i `data/`-mappen
4. Kör notebooken uppifrån och ned

## Frågor vi undersöker

- Är indie-spel bättre betygsatta än andra genrer?
- Har antalet spel på Steam ökat varje år — och har kvaliteten följt med?
- Hur ser prisfördelningen ut bland Steam-spel?
